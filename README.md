## Lesson 3: An Overview of the Third Dimension

### Primitives 三维世界里的图元

A primitive is a single element in a 3D environment, be it a triangle, a line, a dot, or whatever. Following is a list of ways primitives can be combined to create 3D objects.

1. Point Lists 点集合
2. Line Lists 线集合
3. Line Strips 线带, 连起来的线条
4. Triangle Lists 三角形集合
5. Triangle Strips 三角形带, 连起来的三角形
6. Triangle Fans 三角形友元, 共用一个顶点的三角形

### Subtractive Color vs. Additive Color 加色 vs 减色

可以这么理解 自然光的混合是三种颜色加起来等于白色, 这就是加色
但现实中三种颜色的颜料混在一起却得到了黑色,这就是减色
